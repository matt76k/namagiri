# Namagiri: Posit library written in Rust :crab:

![](https://media.giphy.com/media/VbnUQpnihPSIgIXuZv/giphy.gif)

## Overview

Namagiri is a software emulation library for posits.

## Usage

    cargo build --release
    
or

    cargo run --release --example p6e1
